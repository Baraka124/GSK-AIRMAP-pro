🎯 Personalized Action Plan Generator

Zone Definitions

🟢 GREEN ZONE (All Good)

When: CAT ≤ 15, rescue use ≤ 2/week, FEV₁ stable (±5%), no exacerbation symptoms

Actions:

  • Continue regular medications as prescribed
  • Maintain usual activities and exercise
  • Schedule next routine follow-up appointment
  • Continue monitoring symptoms daily

🟡 YELLOW ZONE (Caution)

When: CAT 16-20, rescue use 3-4/week, mild symptom worsening, early signs of exacerbation

Actions:

  • Increase controller medication as per action plan
  • Start rescue medication pack if prescribed
  • Rest more, conserve energy
  • Contact clinic within 48 hours
  • Check inhaler technique
  • Increase monitoring to twice daily

🔴 RED ZONE (Take Action)

When: CAT > 20, rescue use > 4/week, severe symptoms, FEV₁ drop >10%, difficulty breathing

Actions:

  • Start oral steroids if prescribed in action plan
  • Seek medical attention TODAY
  • Go to emergency room if: severe distress, blue lips/nails, confusion
  • Call emergency services if unable to get to hospital
  • Use nebulizer if available and prescribed
